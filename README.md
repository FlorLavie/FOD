#  Fundamentos de Organización de Datos — UNLP

Repositorio con **prácticas y ejercicios** resueltos. 
En este espacio se trabajan los **conceptos y técnicas de organización de datos en memoria externa**, aplicados a la implementación de archivos y estructuras eficientes

##  Temas principales

### 1. Archivos y sistemas de archivos  
- Procesamiento secuencial de archivos.  
- Algorítmica clásica: creación, merge y corte de control.  
- Altas y bajas

### 2. Índices
- Concepto y utilidad de los índices.  

### 3. Árboles
- Árboles binarios de búsqueda (ABB).  
- Árboles balanceados (AVL).  
- Árboles **B** y **B+**: estructura, inserción, eliminación y búsqueda.

### 4. Dispersión (Hashing)
- Concepto de función de dispersión.  
- Densidad de empaquetamiento y colisiones
